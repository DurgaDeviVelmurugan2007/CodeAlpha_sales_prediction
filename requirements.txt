Requirements for sales prediction :

Required Libraries :

1) pandas>=2.0.0
2) numpy>=1.24.0
3) matplotlib>=3.7.0
4) seaborn>=0.12.0
5) scikit-learn>=1.3.0

Roles of Each Libraries Used:

1. pandas
   - Loads and manages the dataset.
   - Performs data cleaning and analysis.

2. numpy
   - Performs numerical and mathematical operations.
   - Used for RMSE calculation and array handling.

3. matplotlib
   - Creates and saves graphs and visualizations.
   - Used for scatter plots and prediction plots.

4. seaborn
   - Creates attractive statistical visualizations.
   - Used for heatmaps, pairplots, and distribution plots.

5. scikit-learn
   - Provides machine learning algorithms.
   - Used for train-test splitting, Linear Regression, predictions, and performance evaluation.

Then anyone can install all required libraries by running:

pip install -r requirements.txt
